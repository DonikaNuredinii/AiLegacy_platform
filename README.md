# 🤖 AI Integration Platform for Legacy Systems

This project is developed as part of a diploma thesis and focuses on integrating Artificial Intelligence (AI) into legacy enterprise systems.

---

## 📌 Overview

The platform demonstrates how modern AI techniques can be integrated with legacy enterprise systems through a scalable and modular architecture.

###  Key Features

* 🔌 API Gateway for system integration
* 🤖 AI-based ticket analysis (TF-IDF + ML models)
* 📊 Monitoring & observability (Prometheus metrics)
* 📁 Legacy system integration (PostgreSQL)
* 🧠 Machine Learning comparison (Logistic Regression vs Naive Bayes)

---

## 🛠️ Tech Stack

### Backend

* Python (FastAPI)
* PostgreSQL
* Docker

### Frontend

* React.js
* Vite

### AI / ML

* Scikit-learn
* TF-IDF
* Logistic Regression
* Naive Bayes

## 🚀 Getting Started

Follow these steps to run the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/DonikaNuredinii/AiLegacy_platform

```

---

### 2️⃣ Start backend and services (Docker)

Make sure Docker Desktop is running, then execute:

```bash
docker compose up --build
```

---

### 3️⃣ Start frontend

Open a new terminal and run:

```bash
cd frontend
npm install
npm run dev
```
